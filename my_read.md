Пошаговая инструкция выполнения домашнего задания: 
1) Реализовать логику валидации запросов к скоринговому API на базе готового «скелета».
2) Сервис на вход принимает POST-запросы с JSON, содержащий набор атрибутов,
требующих проверки на соответствие набору правил, аналогично тому, 
как Django проверяет данные на соответствие описанию формы. 
3) Подразумевается, что реализация будет использовать метаклассы или протокол дескрипторов.
